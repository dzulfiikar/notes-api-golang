# Notes API Golang
    A notes api written in golang

# Features
    * CRUD operations
    * Authentication
    * Authorization
    * Cron Jobs
    * Logger

# Development
    * Install dependencies
    * Run the server using `air`

# Deployment
    1. Bare metal
        * Build the binary using `go build`
        * Run the binary
    2. Docker
        * Make sure to update env values according to your own deployment
